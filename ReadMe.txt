აქტიური ენის (ქართული, ინგლისური) დანაყოფზე მაუსის გადატარებისას, ჩნდება ფანჯარა რომელიც გვაძლევს შესაძლებლობას ავირჩიოთ ენა, გვამისამართებს სერვერის მხარეს, რომელიც არჩეული ენის მიხედვით გვაგზავნის შესაბამისს ლინკის ენდფოინთზე. 

მარჯვენა კუთხეში გვაქვს მენიუ ღილაკი, რომელზე დაჭერისასაც ემატება, მენიუს სხავადასხვა ღილაკები (მეილის, ტელეფონის, ლოკაციის), როცა ეს ღილაკი აქტიურია

მეილის ღილაკზე დაჭერით - ემატება HTML ელემენტები, რომელიც Css ში უკვე დამუშავებულია, და ფოკუსირებას ახდენს ფანჯარაზე, რომელიც გაიხსნა.
